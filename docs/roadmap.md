# Plan реализации проекта

## Название проекта для GitHub

**Основной репозиторий:** `rag-course-platform`

**Подрепозитории:**
- `rag-course-platform/rag-service` - FastAPI сервис
- `rag-course-platform/moodle-block-aiassistant` - Moodle плагин
- `rag-course-platform/course-materials` - Материалы курса

---

## Roadmap

### Этап 1: Базовая инфраструктура (Неделя 1-2)

**Задачи:**
- Создать репозитории на GitHub
- Настроить Docker Compose
- Базовый FastAPI с /health endpoint
- Подключить PostgreSQL и Qdrant
- Интеграция YandexGPT API (тестовый запрос)

**Результат:** Работающее окружение разработки

---

### Этап 2: RAG Pipeline (Неделя 3-4)

**Задачи:**
- Embedder (sentence-transformers)
- Retriever (поиск в Qdrant)
- Generator (YandexGPT с промптами)
- Endpoint POST /ask
- Тестирование на примерах

**Результат:** Работающий RAG без индексации

---

### Этап 3: Индексация документов (Неделя 5-6)

**Задачи:**
- Парсеры PDF/DOCX/TXT
- Chunking стратегии
- Endpoint POST /admin/index
- CLI для индексации
- Загрузка тестовых материалов

**Результат:** Полный RAG с индексацией

---

### Этап 4: Moodle плагин (Неделя 7-9)

**Задачи:**
- Структура block_aiassistant
- Настройки администратора
- UI чата (HTML/CSS/JS)
- PHP API клиент
- AJAX обработка
- Таблица логов

**Результат:** Работающий плагин в Moodle

---

### Этап 5: Безопасность и оптимизация (Неделя 10-11)

**Задачи:**
- Аутентификация по токену
- Rate limiting (Redis)
- Обработка ошибок
- Таймауты
- Логирование
- Кэширование частых запросов

**Результат:** Production-ready система

---

### Этап 6: Контент курса - Модули 1-4 (Неделя 12-13)

**Задачи:**
- Написать лекции (Основы LLM, RAG, Векторы, Подготовка данных)
- Создать тесты
- Подготовить практические задания
- Записать видео (базовый монтаж)

**Результат:** Первая половина курса

---

### Этап 7: Контент курса - Модули 5-8 (Неделя 14-15)

**Задачи:**
- Написать лекции (Retrieval, YandexGPT, Архитектура, Проект)
- Создать тесты
- Разработать финальный проект
- Записать видео

**Результат:** Полный курс

---

### Этап 8: Деплой и тестирование (Неделя 16-17)

**Задачи:**
- Индексация всех материалов курса
- Настройка production сервера
- Установка плагина в Moodle
- Загрузка курса
- Beta-тестирование с пользователями
- Исправление багов

**Результат:** Запущенный курс

---

### Этап 9: Документация (Неделя 18)

**Задачи:**
- README для всех репозиториев
- Инструкции по установке
- API документация (OpenAPI)
- FAQ
- Видео-инструкции для админов

**Результат:** Полная документация

---

### Этап 10: Релиз (Неделя 19)

**Задачи:**
- Финальное тестирование
- Создание релиза v1.0.0
- Публикация в Moodle Plugins Directory
- Анонс курса
- Сбор обратной связи

**Результат:** Публичный релиз

## Приоритеты

**Must Have (MVP):**
- RAG-сервис с базовыми функциями
- Moodle плагин
- Минимум 4 модуля курса

**Should Have:**
- Все 8 модулей
- Статистика запросов
- Кэширование

**Nice to Have:**
- Telegram бот
- Голосовой ввод
- Мобильное приложение

- Uptime > 99%
