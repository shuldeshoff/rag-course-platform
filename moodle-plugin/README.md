# Moodle AI Assistant Block

Блок-плагин для Moodle с интеграцией AI-ассистента.

## Установка

1. Скопировать папку в `blocks/aiassistant/`
2. Войти как администратор
3. Перейти в Site administration → Notifications
4. Следовать инструкциям установки

## Настройка

Site administration → Plugins → Blocks → AI Assistant

- RAG Service URL
- API Token
