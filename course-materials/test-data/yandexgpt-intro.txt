YandexGPT - это семейство больших языковых моделей от Yandex, разработанное специально для русского языка и мультиязычных задач.

Особенности YandexGPT:

1. Оптимизация для русского языка - модель обучена на большом корпусе русскоязычных текстов, что обеспечивает высокое качество понимания и генерации на русском языке.

2. Версии моделей:
   - YandexGPT Lite - быстрая модель для простых задач
   - YandexGPT Pro - более мощная версия для сложных задач
   - YandexGPT Lite RC - экспериментальная версия

3. API и интеграция:
   - REST API для синхронных запросов
   - Поддержка streaming для получения ответов в реальном времени
   - Управление температурой генерации и максимальным числом токенов

4. Безопасность и приватность:
   - Данные обрабатываются в облаке Yandex Cloud
   - Соответствие требованиям хранения данных в РФ
   - Контроль доступа через API ключи

5. Квоты и лимиты:
   - Ограничение на количество запросов в минуту
   - Максимальная длина контекста
   - Тарификация по количеству токенов

Применение YandexGPT:
- Чат-боты и виртуальные ассистенты
- Анализ и суммаризация текстов
- Генерация контента
- Ответы на вопросы
- Интеграция с RAG системами

YandexGPT интегрируется с другими сервисами Yandex Cloud, включая векторные базы данных и системы машинного обучения.
